**概要
複数のドキュメント（またはフォルダ）をドラッグ＆ドロップするだけで
-パッケージの作成
-パッケージとPDFの作成
-PDFの作成
を自動処理します。
http://d.hatena.ne.jp/seuzo/20090212/1234403807


**同梱ファイル2Files

**使用条件
このドロップレットが正常に動作する環境は以下の通りです
-Apple Macintosh
-Mac OS X 10.5.6
-InDesign CS4_J（6.0）


**インストール
このドロップレットを含むフォルダは、どこにおいておいてもかまいません。
適当な場所にファインダーでコピーしてください。


**使用方法
+パッケージ化したい（またはPDF化したい）ドキュメント（複数可）をこのドロップレットにドラッグ＆ドロップします。ドキュメントを含んだフォルダでもかまいません
+どんな処理をするのかを選択するダイアログが出ます。「パッケージのみ」「パッケージとPDF」「PDFのみ」から選んでください。
+「パッケージのみ」「パッケージとPDF」を選んだ場合、パッケージフォルダを保存するフォルダをダイアログで指定します。
+「PDFのみ」を選んだ場合、PDFを保存するフォルダをダイアログで指定します。この時、キャンセルボタンをクリックすると、PDFはドキュメントと同じ階層に作成されます。
+PDFを作成する場合、PDFプリセットがリスト表示しますので、適用したいプリセットを選びます。
+処理が始まります。処理中はIndesignを操作しないでください。
+処理が終了すると、Work was finished.とアナウンスされます。
+正常に処理されたかどうか、確認をおこなってください。


**既知の不具合、またはToDo、あるいは仕様
-InDesign CS3ドキュメントに使用されているフォントがなかったり、リンクが切れていたりした場合、ドキュメントを開いたときに警告しません。パッケージを選んだ場合はプリフライトエラーが発せられますが、続行すればそのままパッケージ、PDF化されます。ご注意ください。
